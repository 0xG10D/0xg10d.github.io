## Quick Start

1. **Clone & Install**
   ```
   git clone https://github.com/0xG10D/g10d.github.io.git
   cd g10d.github.io
   bundle install
   npm install
   ```

2. **Serve Locally**
   ```
   bundle exec jekyll serve
   ```
   Open http://localhost:4000

3. **Deploy**
   ```
   git add .
   git commit -m "Update"
   git push origin main
   ```
   GitHub Pages auto-builds.

## Customization

Edit `_config.yml`:
- `title`, `description`, `avatar`
- `social.links` (GitHub, LinkedIn)
- `theme_mode: dark`

Add posts: `_posts/YYYY-MM-DD-title.md` with frontmatter.

See [Chirpy Wiki](https://github.com/cotes2020/jekyll-theme-chirpy/wiki) for advanced.
